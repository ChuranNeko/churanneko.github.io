# 欢迎来到我的博客！

这是我使用 Next.js 和 Markdown 创建的第一篇博客文章。

## 博客特色

- 🚀 基于 Next.js App Router
- 📝 使用 Markdown 编写文章
- 🎨 现代化的 UI 设计
- 📱 响应式布局
- 🌙 支持深色模式

## 技术栈

这个博客使用了以下技术：

1. **Next.js 15** - React 框架
2. **TypeScript** - 类型安全
3. **Tailwind CSS** - 样式框架
4. **React Markdown** - Markdown 渲染
5. **Gray Matter** - Front Matter 解析
